# สร้างรถถัง

ในบทความนี้ เราจะสร้างรถถังขึ้นมา 1 คัน โดยมีคุณสมบัติดังนี้:

* Animatation ด้วย Sprite
* เดินได้ 4 ทิศ ขึ้น ลง ซ้าย ขวา
* เมื่อเดินถึงขอบฉาก จะไม่สามารถเดินต่อได้

## Sprite คือ ...

Sprite (สำหรับ Computer) ก็คือ ภาพ หรือ Image นั่นแหละ เพียงแต่เวลาเรา Animate รูปเยอะๆ มันจะมี ความกว้าง, ความสูง และ ขนาด Bit Depth หรือ Color Depth เหมือนๆ กันทั้งหมด เลยทำเป็นรูปใหญ่รูปเดียว แบบนี้ครับ [Battle City Sprite](https://www.spriters-resource.com/resources/sheets/57/60016.png)

## AnimatedSprite

สร้าง Scene ใหม่ขึ้นมา เป็น 2D Scene เหมือนกับ battle Scene ที่สร้างในบทที่แล้ว Rename Node2D เป็น Tank และสุดท้าย เซฟในชื่อ tank.tscn ตามรูปที่ 1

[รูปที่ 1](images/normals/02-01.png)

สร้าง Node ใหม่โดยคลิกขวาที่ Tank และเลือก Add Child Node => AnimatedSprite และ Rename เป็น Sprite ตามรูปที่ 2

[รูปที่ 2](images/normals/02-02.png)

จากนั้นทางขวา ตรง Inspector คลิกที่ Frames (ตอนนี้ยัง [empty] อยู่) เลือก New SpriteFrames

คลิกที่ SpriteFrames จะมี Animations ปรากฏที่ด้านล่าง ตามรูปที่ 3

[รูปที่ 3](images/normals/02-03.png)

เราต้องหารูปรถถังมาใช้ และเราคงไม่วาดเอง (เนื่องจากฝีมือวาดรูปผมห่วย T_T) ผมหาไว้ให้แล้ว ไปที่ [Battle City Sprite](https://www.spriters-resource.com/nes/batcity/sheet/60016) และ Download มาครับ

เราต้องการฝีมือ Graphics เล็กน้อย ขอแนะนำโปรแกรม [paint.net](https://getpaint.net) เป็น Photoshop version เล็ก และฟรี Download มาและติดตั้งบนเครื่องคุณเลยครับ

เปิด paint.net ขึ้นมา จากนั้นเลือกไฟล์ Sprite ที่คุณพึ่ง Download มา จะปรากฏรูปที่ 4

[รูปที่ 4](images/normals/02-04.png)

1. เลือก Magic Wand
2. เลือก Tolerance เป็น 0%
3. เลือก สีดำ ในรูป
4. กด Del บน Keyboard
5. จะปรากฏตามรูปที่ 5

[รูปที่ 5](images/normals/02-05a.png)

เซฟไฟล์ไว้ในชื่อ all.png และเอาไว้ใน Folder images ของเกม ตามรูปที่ 6

[รูปที่ 6](images/normals/02-06a.png)

คลิก icon Add Frames from a Sprite Sheet

[รูปที่ 7](images/normals/02-07a.png)

จะปรากฏ Dialog สำหรับเลือกไฟล์ เลือกไฟล์ all.png จากนั้นกำหนด Horizontal: 25, Vertical: 16 แล้วกดปุ่ม Select/Clear All Frames จะได้รูปที่ 8

[รูปที่ 8](images/normals/02-08.png)

คลิก Add 400 Frame(s) จะปรากฏรูปที่ 9

[รูปที่ 9](images/normals/02-09.png)

คลิกที่ Sprite (ซ้าย-บน) แล้วตรง Inspector 
* เลือก Scale แล้วใส่ x = 4, y = 4 จะเห็นรูปรถถังใหญ่ขึ้น
* เลือก Centered แล้วเอาออก จะเห็นรูปรถถังเริ่มจาก (0, 0) แทนที่ (-32, -32)
